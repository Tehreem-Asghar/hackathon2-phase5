# Database Schema Design Skill

This skill provides comprehensive guidance for designing robust database schemas using SQLModel and SQLAlchemy. It includes examples for creating efficient, scalable database models with proper relationships, indexing, and validation that can be applied across different projects.

## Overview

The Database Schema Design skill contains:

- Generic base model templates
- Relationship design patterns
- Indexing and performance optimization techniques
- Data validation and constraint strategies
- Migration best practices
- Common anti-patterns to avoid

## Usage

When designing database schemas for your application, refer to this skill for:

1. Creating base models with common fields
2. Defining proper relationships between entities
3. Optimizing performance with appropriate indexing
4. Implementing data validation and constraints
5. Planning for schema evolution and migrations
6. Following industry best practices for database design
