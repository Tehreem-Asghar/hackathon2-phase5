# Better-Auth Skill

This skill provides comprehensive guidance for implementing authentication using the better-auth library in full-stack applications. It includes examples for both frontend and backend implementations, covering user registration, login, protected routes, and secure session management.

## Overview

The better-auth skill contains:

- Frontend authentication client setup
- Backend authentication API routes
- JWT token management
- User session handling
- Security best practices
- Common troubleshooting tips

## Usage

When implementing authentication in your application, refer to this skill for:

1. Setting up the better-auth client in your frontend
2. Creating backend authentication routes
3. Managing user sessions securely
4. Handling common authentication scenarios
5. Following security best practices
