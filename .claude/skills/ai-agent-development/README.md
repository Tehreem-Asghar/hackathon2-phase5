# AI Agent Development Skill

This skill provides comprehensive guidance for developing AI agents that integrate with your application. It includes examples for creating intelligent agents that can interact with your database, handle user requests, and perform complex tasks using tools.

## Overview

The AI Agent Development skill contains:

- Agent configuration and setup patterns
- Tool creation and integration techniques
- External AI service integration (Gemini/OpenAI)
- Error handling and fallback strategies
- Conversation management approaches
- User-specific agent context management

## Usage

When implementing AI agents in your application, refer to this skill for:

1. Setting up agent configurations with external AI services
2. Creating function tools that interact with your database
3. Managing conversation history and context
4. Handling errors and edge cases gracefully
5. Ensuring proper user isolation in multi-user environments
6. Following security best practices for AI integrations
